# アルゴリズム実技検定
## 学習記録用
目的: 
1. 問題解決能力を鍛える
2. アルゴリズムとデータ構造を学ぶ
3. chatGPTに頼りきりのコピペプログラマにならないようにするため。

### なぜPythonか？
アルゴリズムを学ぶときC++がよく挙げられる。しかし、PythonはC++よりも文法が簡単でアルゴリズムの学習に集中しやすい。
また、普段私がよく使っているRuby、JavaScriptを選択しない理由は、解答例がない場合が多いからである。
「文法が簡単でアルゴリズムの学習に集中できる」「動的型付け言語の中で最もサンプルコードが多い」
の2軸からPythonが得策だと考えた

### 学習計画
アルゴリズム実技検定の参考書とpaiza新卒を利用する
1. アルゴリズム実技検定 公式テキスト
インプット教材。解法とアルゴリズム・データ構造を知る。
2. paiza新卒　プログラミングスキルチェック
アウトプットとして使う。B問題を中心にとにかく数をこなし、コーディング問題に慣れる。
3. JavaScript文法
アルゴリズムを学ぶ上で、最低限知らなければならない。初めて知ったルールや調べたことを、Noteアプリにまとめる。
